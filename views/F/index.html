Research

The result of the research conducted in September was that:

  • Users were able to follow the opt-out form
  • Users were able to understand 2-factor verification
  • Users were able to complete opt-out the process
  • Users felt overwhelmed with the volume of information on the landing page
  • Users felt overwhelmed with the volume of information on the information pages
  • Users found the navigation between pages difficult
  • Users felt the information was too forceful

Design hypotheses

October

  1. If we add a page list to our local header, then we will improve navigation because users will easily be able to find further information about how the NHS handles their data
  2. If we add a navigation element to the bottom of our content pages, then we will improve navigation because users can quickly jump to the next peice of information
  3. If we add a back link to the top of our content pages, then we will improve navigation because users can quickly jump back
  4. If we reduce the width of text in the content pages, then we will improve text comprehension and readability because studies show that text no wider than 600px is easier to read for users
  5. If we add breadcrumbs to the top of each content page, then we will improve navigation because users will be able to just back to the landing page in one click
